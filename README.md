# MeetingX
### 微信小程序-会议管理系统
包含client和server端,server端请在node 4.6环境下运行
服务端开启：进入server目录，运行npm start。
客户端开启：直接使用微信开发者工具打开client即可。
在client中的api.js中可以配置选择访问地址，本人有使用过ngrok内网映射工具，可以用来在开发阶段在手机端测试(没有租服务器*……*)。
